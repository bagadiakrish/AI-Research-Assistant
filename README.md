# AI Research Assistant

An AI-powered PDF Question Answering application built using React + FastAPI + LangChain + Google Gemini.

## Features

- Upload PDF
- AI Question Answering
- Modern React UI
- FastAPI Backend
- Google Gemini Integration


## Tech Stack

- React
- FastAPI
- LangChain
- Google Gemini
- FAISS
- Python
